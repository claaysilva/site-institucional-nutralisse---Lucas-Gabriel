# Site Institucional Nutralisse

Este repositório contém o site institucional desenvolvido para a marca **Nutralisse**.

O projeto foi criado pela **CSCode** (Clayton Silva) como solução digital para apresentar a empresa, seus produtos e canais de contato de forma profissional e voltada à conversão.

> CSCode – Automação e Agentes de IA  
> www.csscode.com.br

## Sobre o projeto

O site foi pensado para funcionar como uma vitrine clara e objetiva da Nutralisse, com foco em:

- **Posicionamento de marca** no nicho de wellness.
- **Clareza na oferta** de produtos (suplementos e conteúdos digitais).
- **Geração de contato** via WhatsApp e e-mail.

A página inicial é composta por:

- **Hero** com banner em destaque, mensagem principal e botões de ação.
- **Seção Sobre** apresentando a Nutralisse e seus diferenciais.
- **Seção de Produtos** com cards para as principais linhas (suplementos e e-books/conteúdos).
- **Rodapé** com dados da empresa, contatos oficiais e links para páginas legais.

Além da home, o projeto inclui:

- **Política de Privacidade** (`politica-privacidade.html`)
- **Termos de Uso** (`termos-uso.html`)

## Tecnologias utilizadas

- **HTML5** para a estrutura das páginas.
- **CSS3** para layout, responsividade e efeitos visuais (gradientes, animações suaves, etc.).
- **JavaScript** puro (`script.js`) para:
  - Scroll suave entre seções.
  - Destaque do item de menu da seção ativa.
  - Animações de entrada conforme o scroll.
- **Google Fonts** (Poppins) para tipografia.
- **Font Awesome** para o ícone do botão flutuante de WhatsApp.

## Estrutura de pastas

- `index.html` – página principal do site.
- `politica-privacidade.html` – página de Política de Privacidade.
- `termos-uso.html` – página de Termos de Uso.
- `style.css` – estilos globais do site.
- `script.js` – comportamentos/efeitos em JavaScript.
- `image/` – imagens utilizadas no layout (logo, banner, etc.).

## Como rodar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/claaysilva/site-institucional-nutralisse---Lucas-Gabriel.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd site-institucional-nutralisse---Lucas-Gabriel
   ```
3. Abra o arquivo `index.html` diretamente no navegador **ou** utilize uma extensão/servidor HTTP simples, como por exemplo:
   ```bash
   npx serve .
   ```

Como é um site estático, não há dependências de backend ou banco de dados.
